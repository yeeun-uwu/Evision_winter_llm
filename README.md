# Evision_winter_llm

이 레포지토리는 Sebastian Raschka의 저서 **[Build a Large Language Model (From Scratch)](https://www.manning.com/books/build-a-large-language-model-from-scratch)** (번역서 [밑바닥부터 만들면서 배우는 LLM](https://product.kyobobook.co.kr/detail/S000217570241))를 학습하며, LLM의 핵심 구성 요소를 밑바닥부터 직접 구현해 본 코드를 보관하는 레포지토리입니다.

## 🚀 프로젝트 개요
- **학습 목표**: 어텐션 메커니즘, 데이터 로딩 파이프라인, GPT 모델 구조 등 LLM의 동작 원리를 코드로 이해합니다.
- **학습 기간**: 2025.12.29 ~ 진행 중
